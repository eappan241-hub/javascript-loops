# Introduction to JavaScript Loops and Array Methods

JavaScript is a programming language used to make web pages interactive. Sometimes we need to work with many values, such as a list of **marks**, **names**, or **prices**. Instead of writing the same code again and again, JavaScript provides **loops** and **array methods**.

## Loops
- **Loops** are used to repeat a block of code multiple times.
- They help reduce repeated code and make programs easier to write and understand.

## Array Methods
- **Array methods** are built-in functions used to work with arrays.
- They make it easier to modify, filter, and combine array data without writing complex loops.
